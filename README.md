# Gender Prediction using 2D:4D Ratio

## 📌 Project Overview
This project predicts gender based on the 2D:4D finger ratio using MATLAB image processing techniques.

## 🛠 Technologies Used
- MATLAB
- Image Processing Toolbox

## ⚙ Methodology
1. Image Acquisition
2. Finger Length Measurement
3. 2D:4D Ratio Calculation
4. Gender Classification

## 📊 Features
- Automatic ratio calculation
- Image-based measurement
- Simple classification logic

## 🚀 How to Run
1. Open MATLAB
2. Run the main.m file
3. Upload finger image
4. View predicted result
